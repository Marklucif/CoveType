La dictée IA sans compromis sur la confidentialité ni le flux de travail.

La saisie vocale cloud peut exposer des propos sensibles, ajouter de la latence, produire un texte brut et perturber les raccourcis. CoveType répond à ces problèmes.

03

04

CONFIDENTIALITÉ ET SÉCURITÉ

Un traitement local facile à comprendre.

La transcription et la reformulation quotidiennes n’appellent aucune API ASR ou de rédaction cloud. L’enregistrement commence seulement lorsque vous maintenez volontairement le raccourci configuré.

L’audio reste local

Les enregistrements temporaires sont traités sur le Mac puis supprimés en fin de traitement.

Les modèles restent locaux

Qwen3-ASR et Qwen3.5 sont stockés dans les données locales de CoveType.

Ni compte ni synchronisation

CoveType ne demande pas de connexion et ne synchronise pas l’historique des transcriptions.

Autorisations limitées

Le microphone capture la voix ; Accessibilité gère le raccourci global et l’insertion dans l’application active.

Quand le réseau est-il utilisé ?

La première installation télécharge l’environnement et les modèles épinglés. macOS télécharge les packs Apple Translation nécessaires et une recherche de mise à jour configurée peut contacter l’adresse de publication. La transcription et la reformulation courantes restent locales.

Statistiques d’utilisation quotidiennes anonymes

Activées par défaut et désactivables dans la fenêtre de retour du client. Au maximum une fois toutes les 24 heures, CoveType envoie par HTTPS un identifiant d’installation aléatoire, la version de l’app, la version de macOS et l’architecture du processeur. Cloudflare détermine seulement le pays ; CoveType ne conserve pas l’IP brute. Aucun audio, transcription ni texte saisi n’est inclus.

ÉTAT DE WINDOWS

L’application actuelle ne prend pas en charge Windows.

Le client dépend d’AppKit/SwiftUI, AVFoundation, Accessibilité, Apple Translation et MLX, propres à macOS. Il ne peut pas être copié ou reconditionné pour Windows.

Qu’est-ce qui fonctionne sous Windows aujourd’hui ?

Le dépôt contient un script Windows 10/11 x64 pour le backend officiel Qwen3-ASR PyTorch et une démo micro dans le navigateur. C’est un test de reconnaissance locale, pas encore une application de saisie globale.

Que faut-il pour une version Windows complète ?

Il faut un client .NET 8 séparé, l’enregistrement WASAPI, les raccourcis globaux et SendInput, un runtime PyTorch/ONNX, une solution de traduction locale, un installateur signé et des tests réels Windows 10/11. Cible pratique : x64, 8 Go de RAM minimum, 16 Go recommandés, 8 à 10 Go libres ; un GPU NVIDIA est facultatif mais réduit nettement la latence.

DÉTAILS

Avant l’installation

CoveType est-il une méthode de saisie macOS complète ?

Non. La version actuelle est un utilitaire vocal de barre des menus. Son raccourci global insère le texte dans l’application active sans changer de source de saisie.

Pourquoi l’autorisation Accessibilité ?

macOS l’exige pour le raccourci global et l’insertion de texte dans une autre application. Vous pouvez la retirer à tout moment dans Réglages Système.

La reformulation locale prend-elle en charge toutes les langues reconnues ?

Le petit modèle Qwen3.5 est multilingue, mais sa qualité n’est pas identique dans les 30 langues. Reconnaissance et qualité de reformulation sont distinctes : testez votre langue pour les usages importants.

Tout est-il open source ?

Le code de l’application CoveType est sous GPLv3. Les modèles Qwen, paquets Python et frameworks Apple gardent leurs propres licences et conditions.
