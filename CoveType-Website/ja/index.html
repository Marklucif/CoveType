現実の課題から設計

プライバシーや操作性を犠牲にしない AI 音声入力。

クラウド音声入力には、機密音声の送信、通信待ち、読みづらい文章、ショートカットの誤作動という課題があります。CoveType はそれらを解決するために作られました。

01

音声のプライバシーが不安？

音声認識と AI 整文は Mac 上で実行され、日常の音声をクラウド文字起こしへ送りません。

02

クラウドの応答待ちが遅い？

ローカルモデルは遠隔 API との往復をなくし、オフラインでも動きます。

03

生の文字起こしが読みにくい？

任意のローカル Qwen モデルが句読点、文区切り、話し言葉を整えます。

04

ショートカットが誤作動する？

キーまたは組み合わせと長押し時間を設定し、Control-C などを通常どおり使えます。

主な機能

クラウドを経由しない音声入力。

01

ローカル文字起こし

Qwen3-ASR 0.6B 8-bit が Apple シリコン上で動作し、30言語を自動検出します。

02

ローカル AI 整文

Qwen3.5 0.8B 4-bit が句読点、改行、話し言葉を整えます。クラウド文章 API は使いません。

03

即時翻訳

Apple Translation とダウンロード済みのオンデバイス言語パックで、話した言語とは別の言語を出力します。

04

カスタム長押し操作

単独キーまたはキーの組み合わせと長押し時間を記録し、通常のショートカットによる誤作動を防ぎます。

05

アプリをまたいで入力

ショートカットを離すと、現在のアプリのカーソル位置に結果を挿入します。

06

メニューバー常駐

小さな呼吸ランプが待機、聴取、処理、完了を表示し、ログイン時に自動起動できます。

プライバシーとセキュリティ

理解できるローカル処理経路。

通常の文字起こしと AI 整文は、クラウド ASR やクラウド文章 API を呼び出しません。設定したショートカットを意図的に長押しした時だけ録音します。

音声はローカルに保持

一時録音は Mac で処理し、処理終了後に削除します。

モデルもローカル

Qwen3-ASR と Qwen3.5 は CoveType のローカルアプリデータに保存されます。

アカウントも同期も不要

ログインは不要で、文字起こし履歴を同期しません。

必要な権限のみ

マイクは音声収集、アクセシビリティはグローバルショートカットと現在のアプリへの文字入力に使います。

ネットワークを使うのはいつ？

初回インストールで固定バージョンの実行環境とモデルをダウンロードします。必要に応じて macOS が Apple Translation の言語パックを取得し、設定済みの更新確認がリリース先へ接続する場合があります。通常の文字起こしと整文はローカルです。

匿名の日次利用統計

既定で有効で、クライアントのフィードバック画面から無効にできます。CoveType は 24 時間に最大 1 回、ランダムなインストール ID、アプリ版、macOS 版、CPU アーキテクチャを HTTPS で送信します。Cloudflare は国だけを判定し、CoveType は生の IP を保存しません。音声、文字起こし、入力テキストは一切含みません。

ローカル音声入力の流れ

  1. 1話す選んだショートカットを押している間だけ録音します。
  2. 2ローカル文字起こしQwen3-ASR が Mac 上で音声を文字にします。
  3. 3ローカル整文任意の Qwen3.5 が端末上で読みやすくします。
  4. 4翻訳または入力Apple オンデバイス翻訳を使うか、アクティブなアプリへ挿入します。

WINDOWS の状況

現在のアプリは Windows に対応していません。

配布中のクライアントは macOS 専用の AppKit/SwiftUI、AVFoundation、アクセシビリティ API、Apple Translation、MLX に依存するため、Windows へコピーや再パッケージはできません。

現在 Windows で使えるものは？

ソースには Windows 10/11 x64 用の公式 Qwen3-ASR PyTorch バックエンド導入スクリプトとブラウザのマイクデモがあります。ローカル認識テストであり、システム全体で使えるトレイ入力アプリではありません。

完全な Windows 版に必要なものは？

別個の .NET 8 クライアント、WASAPI 録音、Windows のグローバルキーと SendInput、PyTorch/ONNX 実行環境、ローカル翻訳の代替、署名付きインストーラ、実機テストが必要です。目安は x64、最低8 GB・推奨16 GB RAM、空き8〜10 GB。NVIDIA GPU は任意ですが遅延を大きく改善します。

詳細

インストール前に

CoveType は macOS の完全な入力メソッドですか？

いいえ。現在はメニューバー型の音声入力ツールです。グローバル長押しショートカットで現在のアプリへ文字を挿入するため、入力ソースの切り替えは不要です。

アクセシビリティ権限が必要な理由は？

macOS ではグローバルショートカットと別アプリへの文字入力にこの権限が必要です。システム設定からいつでも取り消せます。

ローカル整文は全認識言語に対応しますか？

Qwen3.5 小型モデルは多言語ですが、30認識言語すべてで品質が同じではありません。認識対応と整文品質は別なので、重要な用途では事前に確認してください。

すべてオープンソースですか？

CoveType のアプリコードは GPLv3 です。Qwen モデル、Python パッケージ、Apple フレームワークにはそれぞれのライセンスと規約が適用されます。
