실제 문제를 위한 설계

개인정보와 작업 흐름을 희생하지 않는 AI 받아쓰기.

클라우드 음성 입력은 민감한 음성 전송, 네트워크 지연, 정리되지 않은 문장, 단축키 충돌을 만들 수 있습니다. CoveType은 이런 문제를 해결합니다.

01

음성 개인정보가 걱정되나요?

음성 인식과 AI 다듬기는 Mac에서 실행되어 일상 음성을 클라우드 서비스로 보내지 않습니다.

02

클라우드 응답이 느린가요?

로컬 모델은 원격 API 왕복을 없애고 오프라인에서도 작동합니다.

03

원본 받아쓰기가 지저분한가요?

선택적 로컬 Qwen 모델이 문장부호와 구어체를 정리합니다.

04

단축키가 잘못 작동하나요?

키 또는 조합과 누르는 시간을 설정해 Control-C 같은 단축키를 보호합니다.

주요 기능

클라우드를 우회하는 음성 입력.

01

로컬 음성 인식

Qwen3-ASR 0.6B 8-bit가 Apple Silicon에서 실행되며 30개 언어를 자동 감지합니다.

02

로컬 AI 다듬기

Qwen3.5 0.8B 4-bit가 문장부호, 문장 구분과 구어체를 정리하며 클라우드 글쓰기 API를 사용하지 않습니다.

03

즉시 번역

Apple Translation과 다운로드한 온디바이스 언어 팩으로 말한 언어와 다른 언어를 출력합니다.

04

사용자 지정 누르고 말하기

단일 키나 조합키와 누르는 시간을 기록해 일반 단축키가 녹음을 작동시키지 않게 합니다.

05

모든 앱에서 입력

단축키를 놓으면 현재 앱의 커서 위치에 결과를 삽입합니다.

06

메뉴 막대 상주

작은 숨쉬기 램프가 대기, 듣기, 처리, 완료 상태를 표시하며 로그인 시 자동 실행할 수 있습니다.

개인정보와 보안

이해하기 쉬운 로컬 처리 경로.

일상적인 음성 인식과 AI 다듬기는 클라우드 ASR 또는 글쓰기 API를 호출하지 않습니다. 설정한 단축키를 의도적으로 누를 때만 녹음이 시작됩니다.

오디오는 로컬에 유지

임시 녹음은 Mac에서 처리되고 작업이 끝나면 삭제됩니다.

모델도 로컬에 유지

Qwen3-ASR과 Qwen3.5는 CoveType의 로컬 앱 데이터에 저장됩니다.

계정과 동기화 없음

로그인이 필요 없으며 인식 기록을 동기화하지 않습니다.

필요한 권한만 사용

마이크는 음성 수집, 손쉬운 사용은 전역 단축키와 현재 앱의 텍스트 입력에 사용됩니다.

언제 네트워크를 사용하나요?

최초 설치 시 고정 버전 런타임과 모델을 다운로드합니다. macOS는 필요할 때 Apple Translation 언어 팩을 다운로드하며, 구성된 업데이트 확인이 배포 주소에 연결될 수 있습니다. 일반 인식과 다듬기는 로컬에서 처리됩니다.

로컬 음성 입력 경로

  1. 1말하기선택한 단축키를 누르는 동안만 녹음합니다.
  2. 2로컬 인식Qwen3-ASR이 Mac에서 음성을 텍스트로 변환합니다.
  3. 3로컬 다듬기선택적 Qwen3.5가 기기에서 가독성을 높입니다.
  4. 4번역 또는 입력Apple 온디바이스 번역을 쓰거나 활성 앱에 삽입합니다.

오픈 소스로 설계

믿기만 하지 않고 확인할 수 있는 개인정보 보호.

CoveType macOS 앱 코드는 GPLv3로 공개됩니다. 녹음 시작, 모델 위치, 네트워크 사용, 다른 앱 입력 방식을 검토할 수 있습니다.

WINDOWS 상태

현재 앱은 Windows를 지원하지 않습니다.

배포 클라이언트는 macOS 전용 AppKit/SwiftUI, AVFoundation, 손쉬운 사용 API, Apple Translation, MLX에 의존하므로 Windows에 복사하거나 다시 패키징할 수 없습니다.

현재 Windows에서 가능한 것은?

소스 저장소에는 Windows 10/11 x64용 공식 Qwen3-ASR PyTorch 백엔드 설치 스크립트와 브라우저 마이크 데모가 있습니다. 로컬 인식 테스트일 뿐 시스템 전체 트레이 입력 앱은 아닙니다.

완전한 Windows 버전에 필요한 것은?

별도 .NET 8 클라이언트, WASAPI 녹음, Windows 전역 단축키와 SendInput, PyTorch/ONNX 런타임, 로컬 번역 대체 기능, 서명된 설치 프로그램과 실제 Windows 10/11 테스트가 필요합니다. 목표 사양은 x64, 최소 8GB·권장 16GB RAM, 8~10GB 여유 공간입니다. NVIDIA GPU는 선택 사항이지만 지연을 크게 줄입니다.

상세 정보

설치 전 확인

CoveType은 완전한 macOS 입력 방식인가요?

아닙니다. 현재 버전은 메뉴 막대 음성 입력 도구입니다. 전역 누르고 말하기 단축키로 현재 앱에 텍스트를 넣으므로 입력 소스를 바꿀 필요가 없습니다.

손쉬운 사용 권한이 왜 필요한가요?

macOS에서 전역 단축키 처리와 다른 앱의 텍스트 입력에 필요합니다. 시스템 설정에서 언제든 취소할 수 있습니다.

로컬 다듬기가 모든 인식 언어를 지원하나요?

Qwen3.5 소형 모델은 다국어이지만 30개 인식 언어에서 품질이 동일하지 않습니다. 인식 지원과 다듬기 품질은 별개이므로 중요한 용도에서는 먼저 시험하세요.

모두 오픈 소스인가요?

CoveType 앱 코드는 GPLv3로 배포됩니다. Qwen 모델, Python 패키지, Apple 프레임워크에는 각각의 라이선스와 약관이 적용됩니다.
