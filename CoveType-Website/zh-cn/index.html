本地处理

解决真实问题

摆脱云端 AI 语音输入的隐私与工作流负担。

云端语音输入可能暴露敏感内容、增加网络延迟、输出凌乱文字，并与常用快捷键冲突。CoveType 针对这些痛点设计。

01

担心语音隐私？

语音识别和 AI 润色都在 Mac 本地运行，日常口述不会发送到云端转录服务。

02

不想等待云端返回？

本地模型省去远程 ASR 和写作 API 往返，离线时仍可使用。

03

原始转录太凌乱？

可选本地 Qwen 模型会在输入前改善标点、断句和口语表达。

04

快捷键经常误触？

可录制单键或组合键并设置按住延迟，确保 Control-C 等常用快捷键正常工作。

核心功能

不绕云端的语音输入。

01

本地语音识别

Qwen3-ASR 0.6B 8-bit 在 Apple 芯片上运行，并自动检测 30 种语言。

02

本地 AI 润色

Qwen3.5 0.8B 4-bit 可整理标点、断句和口语表达，不调用云端写作 API。

03

即时翻译

说一种语言，通过 Apple Translation 与已下载的设备端语言包输出另一种语言。

04

自定义按住说话

录制单键或组合键并设置按住时长，避免常用快捷键误触录音。

05

跨应用输入

松开快捷键后，CoveType 将结果插入当前应用的光标位置。

06

菜单栏常驻

小型呼吸灯显示空闲、聆听、处理和完成状态，并可登录后自动启动。

隐私与安全

看得懂的本地处理路径。

日常转录与 AI 润色不调用云端语音识别或云端写作 API。只有主动按住已配置快捷键后才开始录音。

音频留在本地

临时录音在 Mac 上处理，流程结束后清理。

模型留在本地

Qwen3-ASR 与 Qwen3.5 保存在 CoveType 的本地应用数据中。

无需账号与同步

CoveType 不要求登录，也不会同步转录历史。

权限用途有限

麦克风用于采集语音；辅助功能用于全局快捷键和向当前应用输入文字。

什么时候会联网？

首次安装会下载固定版本的运行环境和模型；macOS 会按需下载 Apple Translation 语言包；配置更新通道后，更新检查可能访问发布地址。日常识别与润色仍在本地完成。

匿名每日使用统计

默认开启，可在客户端“使用反馈”窗口中关闭。CoveType 每 24 小时最多通过 HTTPS 发送一次随机安装编号、应用版本、macOS 版本和处理器架构；Cloudflare 仅判断国家，CoveType 不保存原始 IP。绝不包含录音、转录结果或输入文字。

你的本地语音转文字路径

  1. 1说话只有按住所选快捷键时才开始录音。
  2. 2本地转录Qwen3-ASR 在 Mac 上把语音转换为文字。
  3. 3本地润色可选 Qwen3.5 在设备端改善可读性。
  4. 4翻译或输入通过 Apple 设备端翻译，或把结果插入当前应用。

WINDOWS 状态

当前应用不支持 Windows。

当前可下载客户端依赖 macOS 专属的 AppKit/SwiftUI、AVFoundation、辅助功能 API、Apple Translation 与 MLX，不能直接复制或重新打包到 Windows。

现在 Windows 上能用什么？

源码仓库已包含适用于 Windows 10/11 x64 的官方 Qwen3-ASR PyTorch 后端安装脚本和浏览器麦克风测试页。它能进行本地识别测试，但还不是能在任意软件输入的系统托盘客户端。

完整 Windows 版需要什么？

需要另做 .NET 8 桌面客户端、WASAPI 录音、Windows 全局快捷键与 SendInput、PyTorch/ONNX 模型运行环境、本地翻译替代方案、签名安装器，并在真实 Windows 10/11 机器测试。建议目标配置：x64，最低 8 GB 内存、推荐 16 GB，8–10 GB 可用空间；NVIDIA 显卡不是必需，但能明显降低延迟。

详细说明

安装前须知

CoveType 会把语音或转录文字发送到云端吗？

日常语音识别和可选 AI 润色都在 Mac 本地运行。首次安装需要下载模型，Apple 可能下载翻译语言包，更新检查会访问 CoveType 发布地址。

CoveType 可以离线语音输入吗？

可以。运行环境、Qwen 模型和所需 Apple 翻译语言包下载完成后，日常转录与本地 AI 润色无需联网。

CoveType 免费并且开源吗？

是。CoveType 应用源码按 GPLv3 开放；Qwen 模型、Python 软件包和 Apple 框架各自保留原许可。

可以在任意 Mac 应用输入吗？

CoveType 通过 macOS 辅助功能 API 在当前光标处输入，可用于编辑器、浏览器、邮件、聊天、文档和许多开发工具。

CoveType 是完整的 macOS 输入法吗？

不是。它是带全局按住说话快捷键的菜单栏语音输入工具，无需切换系统输入源。

为什么需要辅助功能权限？

macOS 要求获得此权限才能处理全局快捷键并向另一个前台应用输入文字，可随时在系统设置中撤销。

本地 AI 润色对 30 种识别语言效果都一样吗？

不一样。Qwen3.5 支持多语言，但不同语言的润色质量存在差异。识别支持和改写质量是两回事，重要流程请先测试。
