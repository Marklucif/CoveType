# Security Policy

## Supported version

Security fixes target the newest CoveType release and the current `main` branch.

## Reporting a vulnerability

Use [GitHub private vulnerability reporting](https://github.com/Marklucif/CoveType/security/advisories/new). Do not open a public issue for a vulnerability that exposes recordings, dictated text, credentials, local files, or a way to execute code.

Include a concise description, affected version, reproduction steps, and impact. Remove all personal data before submitting. The maintainer will acknowledge a complete report as soon as practical and coordinate disclosure after a fix is available.
